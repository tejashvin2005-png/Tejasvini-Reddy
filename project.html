Comparative Evaluation of J48 and NaiveBayes (Using WEKA)

Purpose: Compare two ML algorithms on a car evaluation dataset to determine the best-performing model.

Technical Depth: Preprocessed data using ReplaceMissingValues, NumericToNominal, Resample. Applied CfsSubsetEval + BestFirst for feature selection. Trained models with 10-fold cross-validation. J48 achieved 100% accuracy; NaiveBayes 96.6%. Evaluated using F1-score, ROC area, and confusion matrices.

Tools Used: WEKA, CSV datasets

Relevance: Demonstrates rigorous model evaluation and interpretability — foundational for research in fairness and ensemble learning.
